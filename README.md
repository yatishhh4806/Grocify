# 🛒 Grocify - Multipage Grocery Website

A modern and responsive multipage grocery website built using **React JS** and **Tailwind CSS**. This beginner-friendly project showcases clean UI design, category-based product filtering, reusable React components, responsive layouts, and smooth user experience for an online grocery platform.

## 🚀 Live Features

* Responsive Navbar
* Hero Section with CTA
* Product Categories
* Product Filtering System
* Multipage Routing
* Reusable Product Cards
* Customer Testimonials Slider
* Process Section
* Discount Banner
* Footer Section
* Fully Responsive Design
* Modern UI with Tailwind CSS

---

# 🛠️ Tech Stack

### Frontend

* React JS
* Tailwind CSS
* React Router DOM

### Libraries Used

* React Icons
* Swiper JS

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yatishhh4806/grocify.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd grocify
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```
